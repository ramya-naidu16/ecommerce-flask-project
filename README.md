# Flask E-Commerce Website

This is a mini e-commerce web application built using Flask.

## Features
- User registration and login
- Product listing with images
- Add to cart functionality
- Cart total calculation using session
- SQLite database

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite

## How to Run
1. Install requirements:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
